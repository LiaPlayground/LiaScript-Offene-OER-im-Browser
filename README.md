# LiaScript-Offene-OER-im-Browser

Kurzvorstellung LiaScript im Vergleich zu Authoring-Tool in klassischen LMS

- 10‑minütiger Input entlang der angehängten Leitfragen (falls zutreffend)

Folien optional

- Wie erfassen Autoren ihre Inhalte in dem System / LMS?
- Wie gut können Autoren gemeinsam an Inhalten arbeiten?
- Welche Standards implementiert das System?
- Welche Export-Formate unterstützt das System?
- Welche Import-Formate unterstützt das System?
- Gibt es Schnittstellen zu OER Marktplaetzen oder ähnlichem?
- Welche Plugins kommen zum Einsatz um Inhalte oder bestimmte erweiterte Inhalte zu erfassen?
- Wie gut sind Export-Formate versionsübergreifend kompatibel?
